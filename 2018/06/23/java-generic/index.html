通配符在泛型中的使用

 通配符的出现，可以使我们更加灵活的把握泛型的限定范围。以 List 为例，有以下有几种情况：

  • List<? extends Number> ：这就意味着使用的泛型，即 T 必须是 Number的子类或 Number类型。
  • List<?> list ： 无类型约束，未知类型的通配符。? 无法替代 T作为一个类型变量，仅仅是说明了我们并不知道 T 的类型。
  • List<? super Integer> list ： 这表示，T 应当是 Integer 的父类或 Integer 类型。
