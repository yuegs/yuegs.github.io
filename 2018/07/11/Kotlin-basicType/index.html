Kotlin基础---基本类型

发表于 2018-07-11 | 分类于 Kotlin |
字数统计: 1,149

 Kotlin，是由俄罗斯圣彼得堡的 JetBrains 团队开发出来的，这个词也来源于俄罗斯圣彼得堡附近的科特林岛。它和 Java 之间的关系如下图所示：

 可以看到，Kotlin 是一种以 Java 平台为运行目标的静态编程语言。因此，你可以替代 Java 使用 Kotlin 进行开发，或者同时使用两种语言。

 本节，先来看看 Kotlin 中的基本类型。

数字

 Kotlin 处理数字的方式和 Java 类似，但又不完全一样。类似的是，Kotlin 也提供了内建类型代表数字。

显式类型转换

 Kotlin 中无法实现类型的自动装换，比如说想要从 int 转成 double，需要调用相应方法实现：

 每个数值类型，都可以进行如下转换：

  • toByte(): Byte
  • toShort(): Short
  • toInt(): Int
  • toLong(): Long
  • toFloat(): Float
  • toDouble(): Double
  • toChar(): Char

位操作

 Kotlin 也支持数值的按位操作。比如：

 所有可进行的操作如下：

  • shl(bits) – signed shift left (Java’s <<)
  • shr(bits) – signed shift right (Java’s >>)
  • ushr(bits) – unsigned shift right (Java’s >>>)
  • and(bits) – bitwise and
  • or(bits) – bitwise or
  • xor(bits) – bitwise xor
  • inv() – bitwise inversion

字符

 对于字符而言，注意，并不能作为数字使用。

布尔值

 布尔类型使用 Boolean 表示，有 true 和 false 两个类型。Boolean 在需要的时候会自动装包。它的操作和 Java 中的一致。

字符串

 字符串使用 String 表示，由字符构成，因此可以通过索引操作：s[i] 访问字符。
