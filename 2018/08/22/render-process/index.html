 еңЁдёҠзҜҮзҝ»иҜ‘дёӯпјҢжҲ‘д»¬зңӢеҲ°пјҢе®һйҷ…дёҠжү§иЎҢз»ҳеҲ¶ж“ҚдҪңзҡ„жҳҜдёӘ CanvasгҖӮдҪҶжҳҜпјҢжҲ‘и§үеҫ—иҝҷзӮ№еҲЁзҡ„иҝҳжҳҜдёҚеӨҹж·ұпјҢж¬ зӮ№ж„ҸжҖқгҖӮCanvas еҸҲжҳҜе“ӘйҮҢжқҘзҡ„е‘ўпјҹеҘҪеҘҮе®ғд»¬жҳҜеҰӮдҪ•з»ҳеҲ¶зҡ„гҖӮ
 дәҺжҳҜд№ҺпјҢеҗ„з§Қжүҫиө„ж–ҷпјҢз„¶иҖҢз»Ҳз©¶еҠҹеҠӣдёҚеӨҹпјҢжҖ»и§үеҫ—иҝҳжҳҜдёҚеӨҹж·ұе…ҘгҖӮжң¬зҜҮжҡӮж—¶е°ұжҠҠжүҫеҲ°зҡ„иө„ж–ҷжҖ»з»“дёҖдёӢгҖӮж¶үеҸҠеҲ° AndroidгҖҒFlutter е’Ң ChromeгҖӮдёҠеӣҫдёӯпјҢжғіиҰҒиЎЁиҫҫ з®Ўзәҝ иҝҷж ·дёҖдёӘжҰӮеҝөгҖӮ

Android дёӯзҡ„иҪҜд»¶жёІжҹ“е’ҢзЎ¬д»¶еҠ йҖҹ

softDraw

 йҖҡиҝҮ д»Һ Window еҲ° View иҝҷзҜҮж–Үз« дёӯпјҢжҲ‘д»¬е·Із»ҸеҲҶжһҗиҝҮзЎ¬д»¶еҠ йҖҹе’ҢиҪҜд»¶жёІжҹ“жҳҜеҰӮдҪ•ејҖе§Ӣзҡ„пјҡ

 жӣҙиҝӣдёҖжӯҘе°ҶпјҢеҜ№дәҺиҪҜд»¶жёІжҹ“жқҘиҜҙпјҢView зҡ„ draw(canvas) ж–№жі•зҡ„з»ҳеҲ¶йЎәеәҸеҰӮдёӢпјҡ

  • Step 1, draw the background, if needed
  • skip step 2 & 5 if possible (common case)
  • Step 3, draw the content
  • Step 4, draw the children
  • Overlay is part of the content and draws beneath Foreground
  • Step 6, draw decorations (foreground, scrollbars)
  • Step 7, draw the default focus highlight
  • weвҖҷre doneвҖҰ

 иҖҢз»ҳеҲ¶жң¬иә«пјҢе®һйҷ…дёҠе°Ҷи°ғз”Ё Canvas зҡ„ draw ж–№жі•пјҢиҜҘж–№жі•иҝӣдёҖжӯҘе°ҶиҝӣиЎҢжң¬ең°и°ғз”ЁпјҢSkia е°Ҷд»Ӣе…ҘпјҢжңҖз»Ҳи°ғз”Ё SkCanvas зҡ„зӣёе…і draw ж–№жі•гҖӮ

зЎ¬д»¶еҠ йҖҹ

 2018 е№ҙзҡ„ Google еӨ§дјҡдёҠпјҢеҜ№зЎ¬д»¶еҠ йҖҹз»ҳеҲ¶йғЁеҲҶеҒҡдәҶйқһеёёиҜҰе°Ҫзҡ„д»Ӣз»ҚпјҢе…·дҪ“еҸҜд»ҘзңӢиҝҷйҮҢ Youtube зҡ„и§Ҷйў‘пјҲж–Үз« еә•йғЁеҸҜд»Ҙи§ӮзңӢпјҢиҮӘеӨҮжўҜеӯҗпјүгҖӮзңӢе®Ңд№ӢеҗҺпјҢеҜ№зЎ¬д»¶з»ҳеҲ¶йғЁеҲҶдјҡжңүдёҖдёӘеӨ§иҮҙзҡ„дәҶи§ЈпјҢе°Ҷе…¶ж•ҙзҗҶеҰӮдёӢгҖӮ
 йҰ–е…ҲпјҢжқҘзңӢдёӢе®ғе’ҢеӯҳиҪҜд»¶жёІжҹ“д№Ӣй—ҙзҡ„еҢәеҲ«пјҡ

 еҸҜд»ҘзңӢеҲ°пјҢиҪҜд»¶жёІжҹ“дҫқиө–дәҺ CPUпјӣзЎ¬д»¶жёІжҹ“дё»иҰҒдҫқиө–дәҺ GPUгҖӮдҪҝз”ЁзЎ¬д»¶жёІжҹ“зҡ„еҘҪеӨ„е°ұеңЁдәҺжӣҙеӨҹеҮҸиҪ» CPU зҡ„еҺӢеҠӣпјҢеҗҢж—¶ GPU жӣҙеҠ ж“…й•ҝеӣҫеғҸеӨ„зҗҶгҖӮ
 еҪ“и°ғз”Ё View зҡ„ invalidate() ж–№жі•еҗҺпјҢдҪҝз”ЁзЎ¬д»¶жёІжҹ“зҡ„жү§иЎҢиҝҮзЁӢеҰӮдёӢжүҖзӨәпјҡ

 еҰӮдёҠпјҢжңүеҮ зӮ№йңҖиҰҒиҜҙжҳҺпјҡ

  • жҜҸдёӘ View йғҪеҜ№еә”дёҖдёӘ DisplayList,иҖҢ DisplayList е®һйҷ…дёҠеҚідҪҝеҜ№дёҖзі»еҲ—з»ҳеҲ¶е‘Ҫд»Өзҡ„е°ҒиЈ…гҖӮ
  • зЎ¬д»¶жёІжҹ“ж—¶пјҢеҸӘдјҡжёІжҹ“ dirty viewпјҢиҖҢиҪҜд»¶жёІжҹ“дјҡйҮҚж–°з»ҳеҲ¶жүҖжңүзҡ„ view
  • жңүдәӣиҪҜд»¶жёІжҹ“зҡ„ apiпјҢзЎ¬д»¶жёІжҹ“ж—¶пјҢжҳҜдёҚж”ҜжҢҒзҡ„гҖӮеҚіжңүдәӣ Skia дёӯзҡ„ API еңЁ OpenGL дёӯдёҚйҖӮз”ЁгҖӮ

 дёӢйқўд»ҘдёҖдёӘзӮ№еҮ»дәӢд»¶дёҫдҫӢпјҡ

 еңЁзӮ№еҮ»д№ӢеҗҺпјҢеҸ‘з”ҹдәҶпјҡ

 йңҖиҰҒиҜҙжҳҺзҡ„жҳҜпјҡ

  • UI Thread е°ұжҳҜдё»зәҝзЁӢ
  • RenderThread жҳҜз”ЁдәҺе’Ң GPU жү“дәӨйҒ“зҡ„пјҢз”ЁдәҺиҝӣдёҖжӯҘзҡ„дјҳеҢ–з»ҳеҲ¶е‘Ҫд»ӨгҖҒз”іиҜ· buffer зӯүпјҢдёә GPU зҡ„з»ҳеҲ¶еҮҶеӨҮеҘҪдёҖеҲҮе·ҘдҪңгҖӮ

  • еӣҫеғҸжңҖз»Ҳз”ұ Surface Flinger еңЁ GPU дёӯеҗҲжҲҗгҖӮ

 еҗҲжҲҗзҡ„иҝҮзЁӢдёӯпјҢдјҡйҒҮеҲ°з”ҹдә§иҖ…е’Ңж¶Ҳиҙ№иҖ…жЁЎеһӢпјҡ

 дәӢе®һдёҠпјҢжҜҸдёӘ surface е°ұжҳҜдёҖдёӘз”ҹдә§иҖ…пјҢжҜҸдёӘ surface еҜ№еә”дјҡжңүдёҖдёӘжёІжҹ“ bufferпјҢжңҖз»ҲпјҢз”ұ surface flinger еҗҲжҲҗеӣҫеғҸгҖӮ

Flutter жёІжҹ“иҝҮзЁӢ

 Flutter еӣҫеҪўжёІжҹ“зҡ„з®ЎзәҝеҰӮдёҠжүҖзӨәгҖӮеҗҢж ·зҡ„пјҢUI зәҝзЁӢиҝҳжҳҜдё»иҰҒиҙҹиҙЈзҡ„жҳҜж №жҚ® UI з•Ңйқўзҡ„жҸҸиҝ°з”ҹжҲҗ UI з•Ңйқўзҡ„з»ҳеҲ¶жҢҮд»ӨпјҢиҖҢ GPU зәҝзЁӢиҙҹиҙЈе…үж …еҢ–пјҲеҮ дҪ•ж•°жҚ®з»ҸиҝҮдёҖзі»еҲ—еҸҳжҚўеҗҺжңҖз»ҲиҪ¬жҚўдёәеғҸзҙ пјүе’ҢеҗҲжҲҗпјҲжҳҫзӨәеңЁеұҸе№•дёҠзҡ„з»“жһңпјүгҖӮ

Chrome дёӯзҡ„жёІжҹ“иҝҮзЁӢ

 еҰӮдёҠеӣҫжүҖзӨәпјҢеңЁ DOM ж ‘дёӯпјҢжҜҸдёӘ Node еҜ№еә”жңүдёҖдёӘ RenderObjectпјҢд»ҺиҖҢжһ„жҲҗдәҶдёҖйў—жёІжҹ“ж ‘пјҲRender TreeпјүгҖӮ
 RenderObject зҹҘйҒ“еҰӮдҪ•еңЁ surface дёҠиҝӣиЎҢз»ҳеҲ¶пјҢйҖҡиҝҮдёҖзі»еҲ—еҲ° GraphicsContext зҡ„з»ҳеҲ¶и°ғз”ЁгҖӮGraphicsContext е°ҶеғҸзҙ дҝЎжҒҜеҶҷеҲ°жңҖз»ҲжҳҫзӨәеңЁеұҸе№•дёҠзҡ„bitmap дёӯгҖӮдј з»ҹдёҠпјҢеӨ§еӨҡж•°GraphicsContextи°ғз”ЁйғҪжҲҗдёәеҜ№SkCanvasжҲ–SkPlatformCanvasзҡ„и°ғз”ЁгҖӮ
 жҜҸдёӘRenderObject зӣҙжҺҘжҲ–й—ҙжҺҘйҖҡиҝҮзҘ–е…ҲRenderObject дёҺ RenderLayer зӣёе…іиҒ”гҖӮеҸҜд»ҘзңӢеҲ°пјҢRenderLayer(д№ҹеҸ«еҗҲжҲҗеұӮ) е®һйҷ…дёҠд№ҹз»„жҲҗдәҶдёҖжЈөж ‘гҖӮе®ғжүҖеӯҳеңЁзҡ„ж„Ҹд№үеңЁдәҺпјҢзЎ®е®ҡдәҶжёІжҹ“йЎәеәҸпјҲд»ҺиҖҢдҪҝеҫ— layer д№Ӣй—ҙжңүдәҶиҰҶзӣ–е…ізі»пјүгҖҒеҗҲжҲҗеҚҠйҖҸжҳҺе…ғзҙ зӯүгҖӮжңүи®ёеӨҡжқЎд»¶дјҡи§ҰеҸ‘дёәзү№е®ҡзҡ„ RenderObject еҲӣе»әж–°зҡ„ RenderLayerгҖӮ
 зұ»дјјдәҺRenderObjectдёҺRenderLayersзҡ„е…ізі»пјҢжҜҸдёӘ RenderLayer д№ҹдјҡе…іиҒ”дёҖдёӘ GraphicsLayer гҖӮжҜҸдёӘGraphicsLayerйғҪжңүдёҖдёӘGraphicsContextпјҢз”ЁдәҺз»ҳеҲ¶зӣёе…ізҡ„RenderLayersгҖӮеҗҲжҲҗеҷЁжңҖз»ҲиҙҹиҙЈе°ҶGraphicsContextsзҡ„дҪҚеӣҫиҫ“еҮәз»„еҗҲеҲ°еҗҺз»ӯеҗҲжҲҗиҝҮзЁӢдёӯзҡ„жңҖз»ҲеұҸе№•еӣҫеғҸдёӯгҖӮиҝҷеҗҢж—¶пјҢд№ҹжҳҜдёәд»Җд№ҲдјҡжңүеӨҡдёӘ RenderLayer зҡ„еҺҹеӣ гҖӮеҰӮжһңе…¶дёӯдёҖдёӘ RenderLayer йңҖиҰҒиў«йҮҚз»ҳзҡ„ж—¶еҖҷпјҢд»…д»…йҮҚз»ҳиҜҘ RenderLayer еҚіеҸҜгҖӮ

иҪҜд»¶жёІжҹ“

 дј з»ҹзҡ„иҪҜд»¶жёІжҹ“жҠҖжңҜеҰӮдёӢжүҖзӨәпјҡ

 WebKit д»Һж №йғЁејҖе§ӢйҒҚеҺҶ RenderLayerгҖӮеӨ§йғЁеҲҶе·ҘдҪңжҳҜеңЁRenderLayer :: paintLayerпјҲпјүдёӯе®ҢжҲҗзҡ„пјҢе®ғжү§иЎҢд»ҘдёӢеҹәжң¬жӯҘйӘӨпјҲдёәжё…жҘҡиө·и§ҒпјҢиҝҷйҮҢз®ҖеҢ–дәҶжӯҘйӘӨеҲ—иЎЁпјүпјҡ

  1. зЎ®е®ҡеӣҫеұӮжҳҜеҗҰдёҺжҚҹдјӨзҹ©зӣёдәӨпјҢд»ҘдҫҝжҸҗж—©йҖҖеҮәгҖӮ
  2. йҖҡиҝҮдёәnegZOrderListдёӯзҡ„еӣҫеұӮи°ғз”ЁpaintLayerпјҲпјүпјҢд»ҘйҖ’еҪ’ж–№ејҸз»ҳеҲ¶жӯӨеӣҫеұӮдёӢж–№зҡ„еӣҫеұӮгҖӮ
  3. иҰҒжұӮдёҺжӯӨRenderLayerе…іиҒ”зҡ„RenderObjectsз»ҳеҲ¶иҮӘе·ұгҖӮ
  4. иҝҷжҳҜйҖҡиҝҮд»ҺеҲӣе»әеӣҫеұӮзҡ„RenderObjectејҖе§ӢйҖ’еҪ’RenderObjectж ‘жқҘе®ҢжҲҗзҡ„гҖӮжҜҸеҪ“жүҫеҲ°дёҺдёҚеҗҢRenderLayerе…іиҒ”зҡ„RenderObjectж—¶пјҢйҒҚеҺҶе°ұдјҡеҒңжӯўгҖӮ
  5. йҖҡиҝҮдёәposZOrderListдёӯзҡ„еӣҫеұӮи°ғз”ЁpaintLayerпјҲпјүпјҢйҖ’еҪ’ең°з»ҳеҲ¶жӯӨеӣҫеұӮд№ӢдёҠзҡ„еӣҫеұӮгҖӮ

 еңЁжӯӨжЁЎејҸдёӢпјҢRenderObjectsйҖҡиҝҮеҗ‘еҚ•дёӘе…ұдә«GraphicsContextпјҲйҖҡиҝҮSkiaеңЁChromeдёӯе®һзҺ°пјүеҸ‘еҮәз»ҳеҲ¶и°ғз”ЁпјҢе°ҶиҮӘиә«з»ҳеҲ¶еҲ°зӣ®ж ҮдҪҚеӣҫдёӯгҖӮ

зЎ¬д»¶еҠ йҖҹжёІжҹ“


 е®ғйҖҡиҝҮе®ўжҲ·з«Ҝ - жңҚеҠЎеҷЁжЁЎеһӢе·ҘдҪңпјҢеҰӮдёӢжүҖзӨәпјҡ

  • еңЁе®ўжҲ·з«ҜпјҲеңЁжёІжҹ“еҷЁжҲ–еҶ…зҡ„NaClзҡ„жЁЎеқ—иҝҗиЎҢзҡ„д»Јз ҒпјүпјҢиҖҢдёҚжҳҜзӣҙжҺҘеҸ‘еҮәе‘јеҸ«еҲ°зі»з»ҹзҡ„APIпјҢдёІиЎҢеҢ–е®ғд»¬е№¶жҠҠе®ғд»¬еңЁдёҖдёӘзҺҜеҪўзј“еҶІеҷЁпјҲеңЁе‘Ҫд»Өзј“еҶІеҢәпјүй©»з•ҷеңЁе…¶иҮӘиә«дёҺжңҚеҠЎеҷЁиҝӣзЁӢд№Ӣй—ҙе…ұдә«еӯҳеӮЁеҷЁгҖӮ
  • иҜҘжңҚеҠЎеҷЁд»Һе…ұдә«еҶ…еӯҳжӢҝиө·зі»еҲ—еҢ–е‘Ҫд»ӨпјҢеҲҶжһҗе®ғд»¬е№¶жү§иЎҢзӣёеә”зҡ„еӣҫеҪўи°ғз”ЁпјҲд»Ҙиҫғе°‘йҷҗеҲ¶зҡ„жІҷз®ұпјҢе…Ғи®ёи®ҝй—®иҜҘе№іеҸ°зҡ„3D APIзҡ„иҝҗиЎҢGPUеӨ„зҗҶпјүгҖӮ
