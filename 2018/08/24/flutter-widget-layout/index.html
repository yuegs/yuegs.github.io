Step 1：图解布局

  • 确定行和列
  • 布局中包含网格（grid）吗？

  • UI 中需要 tab 吗？
  • 注意对齐方式、padding 和 border

 首先，来确定最大的元素，在这个实例中，有四个元素排成一列：一个 image，两个 row ，一个 text 块。

 第三行，是按钮部分。这一部分有三个 children : 每个 child 是一个列，包含了 icon 和text。

 一旦布局被图解以后，就很容易采用自下而上的方法实现它。为了避免过深的嵌套，可以使用一些变量和 function。

布局方法

观点：

  • widget 是用来构建 UI 的类
  • widget 被用来布局和 UI 元素
  • 复杂的 widget 是由简单的 widget 组合而成的

 Flutter 布局机制的核心是 widget。在 flutter 中，几乎所有东西都是 widget，比如 image、icon、text 等所有你在 Flutter app 中看到的都是 widget。但是，你看不到的，例如 row、column、grid 等这些用来排列、约束、对齐可见 widget 的其实也是 widget。

对 widget 进行水平和竖直排列

 一个非常普遍的布局方式，就是进行水平或者竖直布局。此时，你可以使用 Row 进行水平布局；使用 Column 进行垂直布局。

观点：

  • Row 和 Column 是两种非常常用的布局方式
  • Row 和 Column 都可以用来出来一列 widget
  • child widget 本身也可以是 Row 或者 Column，甚至更加复杂的 widget
  • 可以使用 Row 和 Column 来指定 children 的排列方式，水平排列或者竖直排列
  • 可以指定 children 如何使用 Row 和 Column 中的可用空间

 为了创建一个 Row 或者 Column ，你需要添加一个 list 的 children widget 到 Row 或者 Column 中。当然， child 本身可能就是一个 Row 或者 Column。下面的示例将向你展示这种嵌套。

对齐 widget

 可以使用 mainAxisAlignment 和 crossAxisAlignment 属性来控制 Row 和 Column 中 children 的对齐方式。对于一个 Column，主轴是竖直的，交叉轴是水平的。

  MainAxisAligment 和 CrossAxisAlignment 这些类提供了控制排列的一些常量。
 在下面的实例中，提供了三个 100像素宽的图片，屏幕块大于 300 像素，因此设置主轴怕屁徐为 spaceEvenly ，这将对每个 image 均分多余出来的水平空间。

 同样的，Column 的工作方式也是如此。如下面示例所示，按照列展示三张图片，每一张有 100 像素那么高。屏幕的高度大于 300 像素，因此当设置主轴排列为 spaceEvenly 时，将平分剩余的竖直空间。

 注意： Row 和 Column 的在主轴上的尺寸可能会超过屏幕大小，这时候，可以对 child widget 使用 Expanded 包裹，Expanded 会自动适配大小，进行合理缩放。如果是多个 Expanded，会按照设定的比例合理适配屏幕。

为 widget 设置大小

 可能会有这样的需求：一个 widget 的大小，是其他兄弟 widget 的两倍。你可以使用 Expanded widget 来控制 widget 在主轴上的大小。Expanded 有 flex 属性，这个属性时一个 int 值，决定了它的填充比例。默认该值是 1 。

压缩 widgets

 默认情况下，Row 和 Column 会尽量的占据主轴上的空间。但是，如果你希望 children widget 能够紧密的挨在一起，可以将 mainAxisSize 设置为 MainAxisSize.min 。

常用的 layout widget

 Flutter 拥有丰富的 layout widget 库，这里列出一些常用的。此处的意图在于能够帮助你尽快的上手，而不是一个完成的 layout widget 列表。更多 widget ，可以参见 Widget Overview,或者在 API reference docs 中搜索。
 下面的 widget 分为两大类：来自于 widgets library 中标准 widget 和来自于 Material Components 中的 widget。

标准 widget

  • Container
     可以对一个 widget 设置 padding、margin、border、背景色等
  • GridView
     将 widget 放置在一个可以滑动的网格中
  • ListView
     将 widget 放置在可以滑动的 列表中
  • Stack
     相当于一个帧布局

Material Components

  • Card
     将 child 放入一个卡片中，这个卡片有圆角，并且可以投射阴影。
  • ListTile
     包含尾部 icon 和头部 icon，中间是 3 行 text(可以缺省)
