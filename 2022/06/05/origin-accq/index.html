初识 DAO

2022-06-05

DAO(decentralized autonomous organization) 的出现，允许任何在网络中的协作关系，以协议和条款的形式而存在，决策随着组织内共识而被触发自动执行。

和传统的中心化组织不同，DAO 具有更加广阔的应用场景和领域，也更加 透明、公平和去中心化，更强调群体的权益和决策权。 这里，没有使用完全去中心化，是因为即使在区块链的网络中，组织内的权力也容易产生集中，组织中的个体有限（远不及区块链中节点的个数），且个体所拥有的权益通常是不一样的（通常，区块链上验证节点的验证权重都是一样的）。

1. DAO Tokenomics

token 在 DAO 中具有至关重要的作用，是 DAO 内各种属性中量化的体现，就像其内部的血液。
token 的流动，使得内部的属权、利益、名誉、决策等运转起来，具有了活力。
DAO 内部，如何通过 token 使得各个属性达成一个正向的循环，相互增强，盘旋上升，最终达到一个平衡，是十分重要的。

对于 token 来说，这将涉及到：

  • 发行量：发行量多少，影响着初始价格和用户心理，不是越多越好，反之亦然。早期，比如 shiba 发行量在 一千万亿，比较随意，所以，它现在比较在意销毁的问题；当前，比如 x2y2，发行量为 一亿。
  • 代币分配：决定了代币的流向，可能的流向包括：投资人、团队、空投、合作伙伴、早期参与人、用户激励池、LP pair。
  • 解锁时间的分配：细水长流，分配的代币，通常会有一个发放周期。
  • 激励策略：在为组织作出贡献时，合理量化激励是非常重要的。
  • 贡献管理：贡献结构应当清晰和完备，为量化激励提供依据。
  • 名誉系统：贡献，可以提升影响力。投票权，应该不仅仅由单个 token 持有量（经济）做唯一参考。
  • 国库管理：token 的盈利，该如何管理，捐赠，投资，风险弥补，或者普天同庆等等。
  • 交易管理：利益的分配和交换，增强流动性。
  • 决策机制：声誉和权利，如何对决策产生影响。

token 的外循环，让 DAO 和更广大的经济体系联系起来，成为其一部分，形成经济互补。

当 DAO 的价值得到体现，价值增强价格，最终的最终，会和法币交汇，实现外循环。个体可以通过兑换，获取到现实中的利益。

对于 token 来说，这将至少应涉及到：

  • 质押收益：将 token 质押给平台，由平台发放红利，质押收益，可能会跟 token 自身和稳定币相关。
  • lp pair：在平台上，与稳定币形成流动性提供者，将 token 与外界的沟通建立起来。
  • swap：token 兑换。

任何一种循环，都不应该失衡，否则，市场会将其调节到另外也一个平衡。当然，这种波动，特别是在稳定以后，可能会对整个组织带来毁灭性打击。


2. 可量化的贡献

当个体对组织提供了支持，如何量化这种贡献，是个复杂的问题。
比如，某个提案被采纳，执行后产生的 影响力 如何被评判；作为开发者，构建了 DAO 合约，应当获得多大的贡献；对于早期的参与者，在提供流动性时的贡献。如下图所示，所有参与到 DAO 建设中的每个角色，其为 组织 的贡献也不同。

为了体现贡献的公开性和透明性，规则应体现在合约中，但合约本身，并不适合过于复杂的量化系统。
或许，一种可行的方案，是项目方在初始时定义若干贡献规则，然后将规则的维护交由组织本身来完善。
完整的权力在民，应是在民众能够明白权力之后。

贡献量化的背后，是名誉和利益的分配。


3. 名利

名，名誉；利，利益。
名和利之间，是一种制衡关系。避免一方过于强大后，造成权力的集中化/中心化。德高望重的人，未必是好人；富甲一方的人，也未必是坏人。
过于强调利益，容易导致通过经济集权；过于强调名誉，容易导致利益的不可理分配，也可能会对 token 的流通性带来一定阻滞。
名誉的来源，主要来自对 组织 的建设贡献；利益的背后，是 token 的持有量。两者应共同决定权力。


4. 权力

我们总是希望权力是民主和公平的。然而， 这也是非常难以实现的，即使借助去中心化网络。

事实上，不管 DAO 如何在组织中 孵化 权力，受限于人类的欲望和认知（七原罪），绝对的民主和公平几乎无法实现。或者，我们可以只能称之为民主和公平的进步。

即使在 DAO 中，我觉得权力应该也是逐步释放到个体的。个人和组织，对于权力都需要有一个认识和适应的过程。


5. 决策

决策，涉及了贡献量化/利益分配/名誉分配/权力管理等。

  • 贡献量化，主要是每种组织内的活动，贡献的奖惩制度等。
  • 利益分配，包括国库资金的流向，风险补偿，质押奖励等。
  • 名誉管理，名誉的用途，名誉的奖惩制度等。
  • 权力管理，涉及利益的权力权重、名誉的权力权重、权力形成决策的过程。
